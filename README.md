# 📔 小小记账本 V2（Streamlit 多页面版）

一个基于 Streamlit 构建的简单记账应用，支持用户注册登录、多用户记账、分类记录。

---

## 🚀 功能亮点

- 🔐 用户注册与登录
- 📅 支出 / 收入记录添加
- 📂 分类（饮食 / 工资 / 娱乐 等）
- 🗑️ 删除单条记录
- 🧹 一键清空全部记录
- 📄 多页面结构（主页面 + 记账页面）
